## 4. YouTube Hub

### Connection

1. Enter OAuth Client ID
2. Click "Save Client ID"
3. Click "Connect YouTube"
4. Complete device verification
5. Select channel/page

### Features

- Browse recent uploads
- Create project from any video
- Direct upload rendered clips
- Token refresh handling

### Upload Flow

1. Select exported video
2. Set title, description, tags
3. Choose visibility
4. Click "Upload"

## 11. Distribution Center

### Export Settings

- **Platform**: YouTube Shorts, TikTok, Instagram Reels, X, LinkedIn
- **Quality**: 1080p, 1440p, 4K
- **Include Captions**: Yes/No
- **Include Voice Over**: Yes/No

### Export Bundle

Each export creates:
- `rendered.mov` - Final video
- `manifest.json` - Metadata
- `captions.srt` - Subtitles (if enabled)
- `summary.txt` - Project summary

### Actions

- Export Current Project
- Reveal in Finder
- Upload to YouTube

- Preview often

- Use highest quality source
- Enable noise reduction
- Match platform aspect ratio
